# Dashboard GAVI

Portal de dashboards para GAVI, desarrollado por **Crezka Accounting Firm**.

## Estructura del proyecto

```
GAVI/
├── index.html        # Dashboard principal
├── images.png        # Logo GAVI
├── css/
│   └── styles.css    # Estilos y colores de marca
└── js/
    └── main.js       # Lógica y fecha dinámica
```

## Tecnologías

- HTML5
- CSS3 (variables, grid, flexbox)
- JavaScript (vanilla)

## Colores de marca

| Color   | Hex       |
|---------|-----------|
| Teal    | `#00706B` |
| Naranja | `#C4622D` |

## Uso

Abre `index.html` en el navegador o despliégalo en cualquier servidor estático.
